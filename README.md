# 🚨 Financial Fraud Detection & Risk Analytics Dashboard

An end-to-end Data Analytics project focused on identifying fraudulent financial transactions and analyzing customer risk behavior using Excel, SQL, Python, and Power BI.

## 📌 Project Overview

Financial fraud is one of the biggest challenges in the digital payment ecosystem. This project analyzes transaction data to detect suspicious activities using behavioral patterns such as transaction frequency, location mismatches, device changes, and amount anomalies.

The objective is to identify potentially fraudulent transactions, classify customer risk levels, and provide actionable insights through interactive dashboards.
